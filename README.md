# 買い物弱者送迎支援プラットフォーム「いきいきの輪」

### 目的

「いきいきの輪」プラットフォームとは、買い物弱者と周りのメンバーとの送迎マッチングシステム。
病院や買い物だけでなく、コミュニティ活動場所への送迎等を通じていきいきとした生活が送れますように。
[詳細→こちら](https://www.code4ikoma.org/?p=398)

### 全体構成

1) Twilioで支援必要者からの電話を受けプッシュボタン方式（IVR：Interactive Voice Response)で用件を入力してもらう
2) 上記内容をherokuへhttpで渡す
3) herokuからkintone DBへ保存
4) herokuからメールとLINE push通知とSendgridメールで送迎ボランティア会員へ一斉配信
5) 送迎ボランティア会員からLINE返信あれば、スケジュールへ自動登録
6) 支援必要者へTwilio電話で通知

詳細はdocフォルダ内のシステム構成図を参照ください。

### 設定

本ソースコードはheroku上のサーバーアプリです。

●SEARCH_TEL_NUMBER_PREFERED　フラグについて 
* 1＝送迎依頼者の電話番号優先(送迎依頼者による入力番号より発信者番号を優先する。発信者番号登録が無ければ、送迎依頼者による入力番号を利用する) 
* 0＝送迎依頼者の入力番号のみを利用
